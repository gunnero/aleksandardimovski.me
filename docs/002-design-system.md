# Design system

The system uses a neutral slate foundation, a restrained blue accent, editorial cards, subtle borders, and typography-led hierarchy. CSS variables define light and dark themes. Layout containers top out at 1180px and remain usable from 390px upward.

Interactive controls have visible focus, navigation is keyboard accessible, landmarks and headings are semantic, and reduced-motion preferences disable decorative movement. Components include the layout, header, footer, page hero, project card, buttons, tags, notices, timeline, and contact panels.
