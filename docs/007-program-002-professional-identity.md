# Program 002: Professional Identity, Resume, and Evidence

Program 002 replaces provisional career copy with verified contact details, chronology, education, languages, and defensible certifications. It adds Kalveri as active company-building work, strengthens evidence-led case studies, publishes the first practical article, and provides matching HTML/PDF resumes.

The work stays factual: overlapping roles are shown by date without inventing employment type, active products are labelled as active development, and private client work remains sanitized. Deployment is explicitly outside this program.

## Acceptance

- Recruiter-readable professional identity and chronology
- Two-page, selectable, print-safe A4 PDF
- Published evidence-based article
- Final 1200×630 social image
- Confidentiality, security, accessibility, metadata, responsive, and quality-gate verification
